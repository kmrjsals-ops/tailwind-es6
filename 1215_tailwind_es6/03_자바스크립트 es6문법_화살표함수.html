자바스크립트 es6문법_화살표함수

함수란? 일련의 내용을 작성하여 사용자가 필요할 때 호출하여 내용을 실행하는 실행문이다.

함수는 매개변수가 있는 함수와 매개변수가 없는 일반 함수가 있다.

select, option 태그 사용시 this는 window객체를 가리키기 때문에 화살표 함수를 사용하지 않는 것이 좋다. 화살표 함수로 하려면 e.target.value를 사용

1. 기존 함수 작성과 실행

		//1번 방식 - 매개변수 있는 함수
		function 함수명(매개변수1, 매개변수2....){
			실행할 내용;
		}

		//2번 방식 - 매개변수 없는 함수
		function 함수명(){
			실행할 내용;
		}

		//3번 방식
		const 변수명 = function(){
			실행할 내용;
		}

		//함수 실행은 함수명을 호출하여 실행한다.
		함수명();

es6문법에서 새롭게 추가된 화살표 함수 작성과 실행방법

		const 변수명 = ()=> {
			실행할 내용;
		}

매개변수가 한개인 유형

		함수명 = 매개변수 => {표현식};
		함수명 = (매개변수) => {표현식};

매개변수가 한개일 경우는 굳이 담지 않아도 된다. 괄호()를 안써도 상관없다.

화살표 함수와 this 키워드

  • this(이것) - 해당 키워드가 속한 객체를 참조하고자 할 때
  • 혼자 써있다면 전역 객체를 의미
  • 함수에서는 전역 객체를 의미
  • 메서드 내부에서는 메서드를 소유한 객체를 의미
  • 이벤트는 이벤트를 받는 객체 의미
